# Zed PIO Assembly

Adds [Raspberry Pi's PIO](https://www.raspberrypi.com/news/what-is-pio/) [assembly language](https://github.com/raspberrypi/pico-sdk/tree/master/tools/pioasm) syntax to the [Zed editor](https://zed.dev/).

Uses [tree-sitter-pioasm](https://github.com/leo60228/tree-sitter-pioasm).
